## 🛠️ Contenido de la Tarea 6
En este repositorio se completaron los siguientes objetivos prácticos:
1.  **Personalización de Plantilla:** Modificación de archivos base (`index.html`, estilos y `README.md`) registrando commits independientes.
2.  **Publicación Web:** Despliegue del sitio en producción usando **GitHub Pages**.
3.  **Rebase Interactivo:** Reorganización y combinación de commits (`rebase -i HEAD~3`) para mantener un historial limpio.
4.  **Autenticación Segura:** Configuración y uso de un **Personal Access Token (PAT)** para el envío de cambios al repositorio remoto.
