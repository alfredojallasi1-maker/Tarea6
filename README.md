# 🎓 Curso de Git y GitHub - UPEA

¡Bienvenido al repositorio oficial del curso práctico de **Git y GitHub**! Este espacio está diseñado para estudiantes y entusiastas de la tecnología de la **Universidad Pública de El Alto (UPEA)** que desean dominar el control de versiones, la gestión de código colaborativo y el despliegue de proyectos web en producción.

---

## 🚀 Objetivo del Curso
Capacitar a los estudiantes en el uso profesional de Git como herramienta de control de versiones local y GitHub como plataforma colaborativa y de despliegue, permitiéndoles integrarse eficientemente a flujos de trabajo de desarrollo de software modernos.

---

## 📚 Contenido del Proyecto
Este repositorio contiene las prácticas, laboratorios y proyectos desarrollados durante las sesiones, estructurados de la siguiente manera:

*   **📂 Conceptos Básicos:** Inicialización de repositorios, gestión del *Staging Area* (`add`), creación de confirmaciones (`commit`) y visualización del historial (`log`).
*   **🌿 Flujo de Ramas:** Creación, cambio y fusión de ramas (`branch`, `switch`, `merge`), junto con la resolución manual de conflictos de código.
*   **🏷️ Gestión de Versiones:** Uso de etiquetas estáticas y anotadas (`tags`) para marcar hitos y versiones estables de software (e.g., `v1.0.0`).
*   **☁️ Sincronización Remota:** Conexión de repositorios locales con la nube (`push`, `pull`, `fetch`) y autenticación segura mediante **Personal Access Tokens (PAT)**.
*   **🛠️ Flujos Avanzados:** Optimización del historial de commits mediante **Rebase Interactivo** (`rebase -i`) para mantener un árbol de cambios limpio y legible.
*   **🌐 Despliegue Continuo:** Publicación automática de sitios web estáticos utilizando **GitHub Pages**.

---

## 🛠️ Requisitos e Instalación
Para replicar y trabajar con este proyecto en tu entorno local, asegúrate de contar con las siguientes herramientas:

1.  **Git:** Descarga e instala la última versión desde [git-scm.com](https://git-scm.com/).
2.  **Consola recomendada:** Se sugiere utilizar **Cmder** o la terminal integrada de tu sistema operativo.
3.  **Editor de código:** [Visual Studio Code](https://code.visualstudio.com/) o tu editor preferido.
